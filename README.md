# Haskell-Project

## Install
1. Install Haskell https://www.haskell.org/downloads/
2. Install required dependencies
```sh
cabal install gloss
cabal install random
```
3. Compile
```sh
ghc main.hs
```
4. Run 
```sh
./main
```
5. Left and right arrow keys to control paddle.
